Under the Hood

How MongoMatch Works

Built with MongoDB Atlas Search — the same technology powering global streaming platforms.

⌨️

Type a character

Your keystroke triggers an autocomplete query via the Atlas Search edgeGram tokenizer — returning suggestions in milliseconds.

🧠

Fuzzy matching kicks in

Made a typo? Atlas Search uses Levenshtein distance (maxEdits: 2) to find movies even when spellings differ.

⚡

Lucene index returns results

Results are ranked by a relevance score computed by Apache Lucene's BM25 algorithm, giving you the most pertinent movies first.

🎯

Filter & refine

Post-search filters for genre, decade, and rating narrow down the 21,000+ movies to exactly what you want.

Backend

Node.js + Express

Lightweight REST API with a dedicated endpoint for autocomplete, full-text search, genres, and trending.

Database

MongoDB Atlas

Cloud-hosted database running on AWS Mumbai, serving 21,000+ records from the sample_mflix dataset.

Search Engine

Atlas Search (Lucene)

Custom index with edgeGram autocomplete and full-text fuzzy search — no external search server needed.

Frontend

Vanilla JS + CSS

Zero-dependency frontend. Premium glassmorphism design, debounced inputs, pagination, and smooth animations.
